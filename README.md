# 2021 Fall Capston Design 2 - 음성인식을 통한 주식 자동매매   
   
- audio_test - 브라우저 음성 녹음 예제 코드   
- server - 메인 서버 디렉터리   
- test - 테스트 코드   
- test_front - 프런트 테스트 코드   
   
#### 구현 완료 목록
- 주식 정보 확인   
- 매수/매도 주문   
- 1~10차 매수/매도호가에 매수/매도 주문   
- 특정 종목 전체 매도   
- 예수금/주문가능금액/계좌번호 확인   
- 계좌정보 확인   
- 보유주식정보 확인   
- 단일 보유주식정보 확인   
- 최고상승종목 확인   
- 최고하락종목 확인   
- 미체결 내역 조회   
- 단일 주문 취소   
- 전종목 주문 취소
